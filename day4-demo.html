Day 4 - Product List Demo

This page demonstrates the product list block with pagination (20 items per page).

Total: 25 products. Page 1 shows 20 items, Page 2 shows 5 items.

Features Demonstrated

  • ✓ Displays 20 items per page
  • ✓ Previous/Next pagination buttons
  • ✓ Responsive grid layout (3 columns on desktop, 1 on mobile)
  • ✓ Hover effects on cards
  • ✓ Page counter showing current page and total
  • ✓ Smooth scroll to top on page change

How to Use in Production
